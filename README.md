# Emoji AI MCP Server

Emoji search, text-based suggestions, emoji-to-text conversion, and counting.

## Tools

- **search_emoji**
- **suggest_for_text**
- **emoji_to_text**
- **count_emojis**

## Quick Start

```bash
pip install mcp
python server.py
```
